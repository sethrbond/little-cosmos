# 🌍 Our World

An interactive 3D globe that maps a love story across the world.

## Setup

1. Run the SQL in `src/supabase.js` (the `SETUP_SQL` export) in your Supabase SQL editor
2. `npm install`
3. `npm run dev` for local development
4. Push to GitHub → Vercel auto-deploys

## Tech

- React + Vite
- Three.js for 3D globe
- Supabase for persistent storage
